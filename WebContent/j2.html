Java Programming >> Question 2 : Java Stdin and Stdout

Most Code Up challenges require you to read input from stdin (standard input) and write output to stdout (standard output).

One popular way to read input from stdin is by using the Scanner class and specifying the Input Stream as System.in.

For example:

Scanner scanner = new Scanner(System.in);

String myString = scanner.next();

int myInt = scanner.nextInt();

scanner.close();

System.out.println("myString is: " + myString);

System.out.println("myInt is: " + myInt);

Input Format :

10

20

Output Format :

The sum of 10 and 20 is 30

Sample Output :

The sum of 10 and 20 is 30
