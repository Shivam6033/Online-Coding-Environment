C Programming >> Question 2 : C Stdin and Stdout

Most Code Up challenges require you to read input from stdin (standard input) and write output to stdout (standard output).

One popular way to read input from stdin is by using the scanf() method and specifying the Input Stream as printf()

For example:

int a;

scanf("%d",&a);

printf("The number is %d",a)

Input Format :

10

20

Output Format :

The sum of 10 and 20 is 30

Sample Input :

10

20

Sample Output :

The sum of 10 and 20 is 30
